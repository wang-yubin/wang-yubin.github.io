**知识图谱生成 Agent（大模型应用探索）⭐**

**实体数据治理服务 ⭐**

**NLU 离线工程维护**
接手维护 NLU 离线工程，负责提供 Java SDK 版本，集成离线 NLU 所需的分类模型、NLU 模型及离线快速干预模块。承担多版本的差异兼容工作，保障多车型离线训练流程的稳定迭代，支撑线上 NLU 模型的持续更新。

**线上干预服务**
入职初期独立开发并维护线上干预服务，高峰期日均请求量约 1.6 亿次，QPS 峰值近 20。该服务基于 HanLP 构建，通过规则配置与话术配置实现对线上语音问题的实时干预，将话术映射到对应的 domain / intent / slot 进行修正，覆盖多款量产车型，保障线上语音体验的准确率与及时性。
